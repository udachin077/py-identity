<h1 style="text-align: center">PyIdentity</h1>

### Features:

* **Managers**
    - [x] UserManager
    - [x] RoleManager
* **Models and stores**
    - [x] Supports personal data field (encrypted)
    - [x] SQLAlchemy
    - [x] Tortoise-ORM
* **Password hashers**
    - [x] Bcrypt
    - [x] Argon2
    - [x] PBKDF2
* **Validators**
    - [x] User validator
    - [x] Role validator
    - [x] Password validator

## Installation

```
$ pip install py-identity
```
